# Files to Upload to GitHub (Backend)

## 📁 Upload these folders/files to your GitHub repository:

### ✅ Required Folders:
- `Controllers/` (entire folder)
- `Data/` (entire folder)
- `Migrations/` (entire folder)
- `Models/` (entire folder)
- `Properties/` (entire folder)
- `Services/` (entire folder)

### ✅ Required Files:
- `appsettings.json`
- `Program.cs`
- `railway.json`
- `RamApi.csproj`
- `RamDb.db` (your database with 15 RAMs)

### ❌ Skip These (not needed):
- `bin/` folder
- `obj/` folder
- `.db-shm` files
- `.db-wal` files
